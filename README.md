# FriendShell

## Web Application with open source code 

![Drag Racing](client/public/friendshell.png)

### Description:
#### Web Application where you can openly or anonymously ask anyone about what interests you very much.

- ReactJS+Redux
- ExpressJS
- Mongoose
- JWT-Auth
- 'styled.components'
